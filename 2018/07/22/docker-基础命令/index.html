docker 基础命令

Posted by feelj on 2018-07-22

docker常用命令#

  • 帮助命令
    docker –help
  • docker 信息查看
    docker version
    docker info
  • 查看本地镜像
    docker images [参数]
    • docker images -a：列出本地所有镜像(含中间映像层)
    • docker images -q: 只显示镜像id
    • docker images –digests: 显示镜像的摘要信息
    • docker images –no-trunc:显示完整的进行的信息
  • 镜像搜索
    • docker search [镜像名称]: 根据镜像名称搜索镜像
    • docker pull [镜像名称:TAG] : 下载镜像，默认下载TAG为lastest版本的镜像文件
    • 删除镜像
      • docker rmi [镜像名称或ID:TAG]: 根据名称或id删除镜像(TAG默认是lastest)
      • docker rmi -f [镜像名称或ID:TAG]:强制删除，多个用空格分开
      • docker rmi -f ${docker images -q}: 删除全部

  • 镜像操作命令

    • docker run [OPTIONS] images [COMMAND][ARG…]

      • OPTIONS说明：有一些是一个减号，有的是两个减号
        • –name=”容器名称”：为容器指定一个名称
        • -d：后台运行容器，并返回容器id，也即启动守护进程模式
        • -i：以交互模式运行容器，通常与-t同时使用
        • -t：为容器分配一个伪输入终端，通常与-i同时使用
        • -P：随机映射端口
        • -p：指定映射端口，有四总形式
          • ip:hostPort:containerPort
          • ip:containerPort
          • hostPort:containerPort
          • containerPort
      • 使用docker镜像构架centos系统

      • docker ps [OPTIONS]: 列出所有运行的docker容器
        • OPTIONS说明
          • -a：列出当前所有正在运行的容器+历史上运行过的容器
          • -l：显示最近创建的容器
          • -n：显示最近n个创建的容器
          • -q：静默模式，只显示容器编号
          • –no-trunc：不截断输出、
    • exit: 退出容器，(ctrl+P+Q也行,去别在于exit容器停止退出，ctrl+P+Q容器不会停止退出)
    • docker start [容器名或id] :启动容器
    • docker restart [容器名或id]： 重启容器返回容器编号
    • docker stop [容器名或id]：停止容器
    • docker kill [容器名或id]：强制停止容器
    • docker rm [容器id]：删除已停止的容器
    • docker rm -f [容器id]：强制删除容器(运行的和没运行的)
    • docker rm -f ${docker ps -a -q}:删除多个容器
    • docker run -d [容器名或id]: 启动容器后台运行

    • docker logs -f -t -tail [容器id]:查看容器日志

      docker run -d centos /bin/sh -c "while true; do echo hello zzyy;sleep 2; done"

    • docker top [容器id]: 查看容器进程
    • docker inspect [容器id]: 查看容器内部细节
    • docker attach [容器id]:退出容器后重新进入
    • docker exec -it [容器id] bashshell
